# Projet Maven 

Projet de base créé avec **Maven** pour pratiquer la configuration, la gestion des dépendances et la structure standard d'un projet Java.

---

##  Objectif du projet

Ce projet sert d'introduction  pratique à :
- La création et configuration d'un projet Maven
- La structure standard Maven (`src/main/java`, `pom.xml`)
- L'ajout et gestion de dépendances
- La compilation, les tests et l'exécution via Maven

---

##  Technologies utilisées

- **Java** (JDK 1.0.15)
- **Maven** (outil de build principal)
- (Dépendances possibles :  JUnit, Hibernate/JPA, MySQL Connector... — à vérifier dans pom.xml)
